# Basic Rule-Based Chatbot

## 📌 Description
This project is a simple rule-based chatbot built using Python. The chatbot responds to predefined user inputs such as greetings, questions, and exit commands.

## ✨ Features
- Greets the user.
- Responds to basic questions.
- Handles unknown inputs gracefully.
- Ends the conversation when the user types "bye".

## 🛠️ Technologies Used
- Python 3

## ▶️ How to Run
1. Install Python 3.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run:

```bash
python chatbot.py
```

## 📚 Concepts Used
- Functions
- Loops
- Conditional Statements
- User Input
- String Handling

## 🎯 Outcome
This project demonstrates the fundamentals of conversational programming using simple rule-based logic in Python.
